# アプリの目的

- お金の管理が下手なので機械的に管理したい

# 必要な機能のブレスト

- 現在の貯金額表示
- 毎月の出費リスト
  - 食費代.娯楽代など
- 当月の収入リスト
- 収入-出資で貯金
- 出資額割り当ての時に何％でもできるようにする。
- 出資するもののオプション
- 毎月の貯金額表示
- 毎月の出資額も表示
